# Bundled font

**DejaVuSans.ttf** — used to regenerate watermarked PDFs with a Unicode font so
the zero-width watermark characters survive in the PDF text layer (standard PDF
fonts drop them).

DejaVu Fonts are released under a permissive, free license (a Bitstream Vera
derivative); they may be redistributed and bundled. See the DejaVu Fonts project
for the full license text: https://dejavu-fonts.github.io/
